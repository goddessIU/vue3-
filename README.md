## 项目完成功能：
1. 获取商家列表
2. 获取用户地址
3. 切换用户地址
4. 用户选择地址
5. 对商家进行排序
6. 获取商家详细信息
7. 获取商品评价
8. 购物车功能
9. 下单
10. 获取个人历史订单
11. 登录
12. 用户个人信息
